# ArenaPlay - Modern Sports Venue Booking Platform

![ArenaPlay Hero](/public/images/general/hero_bg.png)

**ArenaPlay** adalah platform booking lapangan olahraga berbasis web modern yang dirancang untuk memberikan pengalaman reservasi yang cepat, mudah, dan premium. Dibangun dengan teknologi terkini, sistem ini memfasilitasi penyewaan berbagai jenis arena olahraga mulai dari Futsal, Basket, Badminton, Padel, hingga Mini Soccer.

*   **Desain Premium & Responsif**: Antarmuka pengguna (UI) yang modern dengan estetika "Dark Mode" yang elegan dan dukungan penuh untuk berbagai perangkat (Mobile, Tablet, Desktop).
*   **Booking Mudah & Cepat**: Alur pemesanan yang disederhanakan dengan integrasi langsung ke WhatsApp Admin untuk konfirmasi instan.
*   **Informasi Terpusat**: Detail lengkap mengenai fasilitas, harga, dan ketersediaan lapangan dalam satu tampilan yang informatif.
*   **Interaktif**: Dilengkapi dengan animasi halus, modal popup detail, dan peta lokasi interaktif.
*   **SEO Friendly**: Struktur kode yang dioptimalkan agar mudah ditemukan di mesin pencari.

## ğŸš€ Fitur & Menu

### 1. **Beranda (Home)**
*   **Hero Section**: Tampilan utama yang memukau dengan CTA (Call to Action) yang jelas untuk booking atau cek jadwal.
*   **Stats Counter**: Menampilkan kredibilitas arena dengan angka statistik (Tim/Minggu, Rating, Jenis Arena).

### 2. **Lapangan (Venues)**
*   **Kategori Filter**: Memudahkan user mencari lapangan berdasarkan jenis olahraga (Futsal, Basket, Padel, dll).
*   **Kartu Lapangan**: Menampilkan foto, rating, harga, dan fasilitas utama.
*   **Detail Modal**: Popup informatif yang menampilkan galeri foto, deskripsi lengkap, fasilitas, dan formulir booking.

### 3. **Jadwal & Event**
*   **Simulasi Jadwal**: Tampilan visual ketersediaan lapangan secara real-time (simulasi).
*   **Upcoming Events**: Daftar turnamen yang akan datang dengan tombol detail untuk melihat info lengkap hadiah dan pendaftaran.

### 4. **Blog & Edukasi**
*   Artikel tips dan trik olahraga untuk memberikan nilai tambah bagi pengguna.
*   Dilengkapi dengan modal "Baca Selengkapnya" agar user tidak perlu berpindah halaman.

### 5. **Membership**
*   Pilian paket member (Regular, Pro, Corporate) dengan perbandingan harga dan keuntungan yang jelas.

### 6. **Fitur Pendukung**
*   **Dark/Light Mode Toggle**: Personalisasi tema sesuai preferensi pengguna.
*   **Scroll to Top**: Tombol navigasi cepat kembali ke atas halaman.
*   **Maps Integration**: Lokasi venue yang terintegrasi dengan Google Maps.

## ğŸ“¦ Instalasi & Menjalankan Project

Ikuti langkah berikut untuk menjalankan project ini di komputer lokal Anda:

1.  **Clone Repository**
    ```bash
    git clone https://github.com/rivanalamsyah/arenaplay.git
    cd arenaplay
    ```

2.  **Install Dependencies**
    Pastikan Node.js sudah terinstall, lalu jalankan:
    ```bash
    npm install
    ```

3.  **Jalankan Development Server**
    ```bash
    npm run dev
    ```
    Aplikasi akan berjalan di `http://localhost:5173`.

4.  **Build untuk Production**
    ```bash
    npm run build
    ```
